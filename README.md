# Pulso Gym Landing

Landing page para un gimnasio de alto rendimiento, construida para comunicar energía, disciplina y conversión directa a WhatsApp.

El proyecto busca que la marca se sienta intensa, clara y accionable desde el primer viewport: propuesta fuerte, CTA visible, servicios ordenados y navegación simple.

---

## Contexto

Pulso Gym necesitaba una presencia digital directa para captar nuevos miembros sin depender de formularios largos ni procesos de contacto lentos. La página prioriza el camino más corto entre interés y consulta.

---

## Mi rol

Diseño UI, desarrollo frontend, copy orientado a acción, estructura responsive, integración con WhatsApp y microinteracciones para reforzar energía de marca.

---

## Decisiones de UX

- Hero de alto impacto con CTA principal above-the-fold.
- WhatsApp como canal principal para reducir fricción.
- Secciones enfocadas en servicios, cultura y prueba de valor.
- Tipografía condensada y jerarquía fuerte para proyectar intensidad.
- Mobile-first para consultas desde smartphone.

---

## Stack

- HTML5 semántico
- CSS3
- JavaScript vanilla
- Responsive design
- Integración directa con WhatsApp
